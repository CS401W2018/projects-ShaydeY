Question N Answers:

Q1: Explain the difference between an absolute link and a relative link. Why would you use relative links?

An absolute link is a complete and total link to any site; within and outisde your domain. A relative link is a link to something else in your domain. Relative links would be used to make it easier to navigate within your own domain. Instead of needing the whole link every time, just the file name (sometimes the folders as well) to direct to is needed.

Q2: List the 3 rules with relative links.

  1. Use the EXACT SAME file name in your link.
  2. Up in directory - lead with "../"
  3. Start at root - start with a "/"

Q3:Do links always need to be inside a NAV tag?

No.
