Questions and Answers

List out the different ways to select an item.

  1. Use tag name (General Use)
  2. Decendant rule (General Use)
  3. ID (Specific - use less; noun)
  4. Class (Specific - use adj to describe)

What is the difference between an ID and a Class?

An Id can only be used once; will never be repeated. A class can name more than one tag and vice versa.

What is the difference between a SPAN tag and a DIV tag?

With a span tag there is no line breaks, whereas with the div tag there is.
