Questions and Answers

Explain the difference between inline, embedded, and external CSS.

Inline CSS - uses the style attribute within a line of code. Don't use; 2nd priority
Embedded CSS - uses the style tag as its own set of code in the header. Uses a CSS Selector to declare what to affect. Don't use
External CSS - putting all CSS in another file. Needs to be named in a file with ".css". Use this one.

Explain why you should only use external CSS.

Seperation of concerns. Helps browsers display things correctly, as well as see/understand intended things. Keeping html code for the info and css code for design seperate. Also for good scalability and cohesiveness. More sites all only need 1 css file.

What is a selector used for?

A selector is used to declare what part of your code (tag) is affected by your CSS code.
