Questions and Answers

Why is the viewport meta tag important?

So that while view the page on your phone, it formats and shows correctly.

How can you adjust the CSS for printing?

Make a new css file specifically for printing; link it and change the media

When would you want to adjust the CSS for printing?

When you know/intend for the page to be printed

Questions Part 2

Why do we need a viewport meta tag?

So that while view the page on your phone, it formats and shows correctly.

When you select the same tag twice, and in one of them you say "flex-direction: column;" and the other says "flex-direction: row;", which one takes priority?

The one that comes later in the CSS takes priority. Like normal flow of things.

How do you change the CSS for a web page displayed on a phone?

Use a @media to set a condition at which different CSS takes priority.
