Questions:

What should you focus on during class and after class?

Making sure your indentation is correct, accurate, and clean. The relationship between parent and children or siblings should be clear and evident. Focus on following along in class to understand, after you can experiment and figure it out more.

What is the main difference between CSS Grid and CSS Flex?

CSS Grid is 2 dimentional, CSS Flex is 1 dimentional. Flex is also used for mobile devices. CSS Grid is an authoritarian parent (i.e exactly as you code). Flex will allow the child to adjust itself accordingly.
