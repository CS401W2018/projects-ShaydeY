Domain Name

A part of the URL which represnts the website group. The URL tells the very specific page you are on, where as the Domain or domain name would tell you the site/companies name.

HTTP

A set of rules that tells you how browsers and websites communicate. It is the core of the Web, and allows access and interactin with the websites.

HTTPS

Essentially the same as HTTP however, the communication has been encrypted and secured.

IP Address

A unique "name" made up of a set of characters. Identifies the specific computer to allow communication with the web.

DNS

Domain Name System. Converts Domain names into IP Adresses. This allows us to not have to remember the specific IP Adresses

W3C Recommendations

These are standards or reccomendations that the organization W3C (or World Wide Web Consortium) creates for the web. Basically what the organization says is the norm or standard for the web.

XML, XHTML, HTML, HTML5

XML provides rules/guidelines for creating structuring, and encoding documents. XHTML is basically HTML and XML combined; that is, it has the same function as HTML, but the same rules as XML docs. HTML is the language for structuring pages e.g. how to display text and etc. HTML5 is just the latest version of HTML.
